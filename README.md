# LLDownloader

Objective-C download framework + demo app.

| Folder | Contents |
| --- | --- |
| [`LLDownloader/`](./LLDownloader) | Library sources + `LLDownloader.podspec` |
| [`LLDownloader-Demo/`](./LLDownloader-Demo) | iOS demo app depending on the library via a local CocoaPods pod |

See the per-folder README for details, API reference, and integration notes.

## Quick start

```sh
cd LLDownloader-Demo
pod install
open LLDownloader-Demo.xcworkspace
```
