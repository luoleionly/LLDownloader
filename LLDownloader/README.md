# LLDownloader (library)

Library sources and `LLDownloader.podspec` for the `LLDownloader` pod.

Full documentation — features, installation, API reference, status lifecycle, code examples, background-session handoff, error model — lives in the **top-level [README](../README.md)** of this repository.

## Contents of this folder

| Path | Purpose |
| --- | --- |
| `LLDownloader.h` | Umbrella header |
| `LLDownloader.podspec` | CocoaPods spec |
| `General/` | Core types (`LLSessionManager`, `LLDownloadTask`, `LLCache`, …) |
| `Extensions/` | Foundation category helpers (`tcl_`-style utilities, hashing, URL coercion) |
| `Utility/` | `LLResumeDataHelper`, `LLFileChecksumHelper` |
| `LICENSE` | MIT license |
